The Five Pillars

1. FastSense Core

Ultra-fast plotting engine with smart downsampling. Renders 10M–100M+ points at 200+ FPS using MinMax/LTTB algorithms and lazy multi-resolution pyramid cache.

2. Sensor Thresholds

State-dependent threshold rules that activate based on system conditions (idle, running, error). Automatic violation detection and grouping.

3. Event Detection

Groups threshold violations into discrete events with statistics (peak, mean, RMS, duration). Real-time streaming detection with live file polling.

4. Dashboard Engine

Widget-based dashboards with 15+ widget types on a responsive grid. JSON/.m persistence, live mode, and visual edit mode with drag & resize.

5. Web Bridge

TCP server bridging MATLAB dashboards to web/Electron frontends. Bidirectional callbacks enable browser-based HTML5 visualization using uPlot.

On zoom/pan, a binary search finds the visible range in O(log N), selects the optimal pyramid level, and downsamples to pixel-density resolution. The pyramid is built lazily — only computed on first access.

Example: 50M points → 500K → 5K (3 pyramid levels, each 100x reduction)

Smart Threshold Monitoring

State-dependent thresholds that adapt to operating conditions:

  • Condition-based rules — thresholds activate only in matching states
  • Industrial alarm bands — HH / H / L / LL with color-coded regions
  • Automatic violation markers — breach points highlighted in real-time
  • Pre-computed during resolve() for instant access

Simple API — 6 Lines to 10M Points

install;  % one-time setup

x = linspace(0, 100, 1e7);
y = sin(x) + 0.1 * randn(size(x));

fp = FastSense('Theme', 'dark');
fp.addLine(x, y, 'DisplayName', 'Sensor');
fp.addThreshold(0.8, 'Direction', 'upper', 'ShowViolations', true);
fp.render();

Drop-in API — works like plot() but handles 10M+ points seamlessly

Technical Highlights

SIMD-Optimized MEX

C MEX kernels with AVX2 (x86) and NEON (ARM) intrinsics for binary search, MinMax, LTTB, and violation computation. Auto-fallback to pure MATLAB.

Disk-Backed Storage

SQLite-backed chunked storage for 100M+ point datasets. Pre-computed L1 MinMax pyramid for instant zoom-out without loading full data.

Cross-Platform

Runs on MATLAB R2020b+ and GNU Octave 7+ across macOS, Linux, and Windows. No toolbox dependencies — fully self-contained.
