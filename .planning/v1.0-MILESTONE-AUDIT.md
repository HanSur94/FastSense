## Integration Findings

### Medium: ReflowCallback not injected in multi-page addCollapsible path

When `addWidget` routes to a multi-page `DashboardPage`, it early-returns before the `ReflowCallback` injection block. This means `addCollapsible` on multi-page dashboards produces collapsible groups that collapse/expand visually but don't trigger grid reflow for sibling widgets.

**Impact:** Layout incorrect after collapse/expand on multi-page dashboards. No data loss or error.
**Fix:** Inject ReflowCallback before the `return` in the multi-page routing branch.

### Low: DividerWidget Thickness/Color lost in .m export

The `.m` export paths (`exportScript`, `exportScriptPages`) emit only `Position` for divider widgets. Custom `Thickness` and `Color` properties are lost. JSON round-trip works correctly.

### Low: Missing combination tests

- addCollapsible + multi-page (no test)
- YLimits + DetachedMirror clone (no test)
- DividerWidget in TestDashboardSerializerRoundTrip property assertions
