**4. Add cache update method:**
Add a private method `updateTimeRangeCache`:
```matlab
function updateTimeRangeCache(obj)
    if ~isempty(obj.Sensor) && ~isempty(obj.Sensor.X)
        x = obj.Sensor.X;
        n = numel(x);
        if n == 0
            obj.CachedXMin = inf;
            obj.CachedXMax = -inf;
            return;
        end
        % Incremental: only check if new data extends range
        % For sorted time arrays, last element is max candidate
        obj.CachedXMax = x(n);
        % Min only changes on full reassignment, not append
        if isinf(obj.CachedXMin)
            obj.CachedXMin = x(1);
        end
    elseif ~isempty(obj.XData)
        obj.CachedXMin = min(obj.XData);
        obj.CachedXMax = max(obj.XData);
    end
end
```

**5. Wire cache updates into update() and refresh():**
- At the end of `update()` (after successful updateData or fallback refresh), call `obj.updateTimeRangeCache()`
- At the end of `refresh()` (after successful rebuild), call `obj.updateTimeRangeCache()`
- In the `render()` method, after `fp.render()`, call `obj.updateTimeRangeCache()` and set `obj.LastSensorRef = obj.Sensor`

**6. Invalidate cache on sensor change:**
