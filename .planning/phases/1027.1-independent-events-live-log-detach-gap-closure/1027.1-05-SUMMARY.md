# Phase 1027.1 Plan 05: Companion Integration Summary

Refactored `FastSenseCompanion.m` from "one log strip" to "two independent log panes": split `LogPane_`/`hLogStateDD_`/`hDetachedLogFig_` into paired Events/Live properties, widened the toolbar to host two dropdowns, added an inner `hLogStripGrid_ = uigridlayout([2 1])` with two `LogPaneRoot`-tagged sub-panels, replaced `setLogState_(newState)` with parameterized `setLogState_(which, newState)`, added `rebalanceLogStrip_` as the single owner of inner+outer row heights, and migrated theme/teardown/forwarding/test-accessor surfaces to the new shape — all with `mh_lint` and `mh_style` clean.

### `setLogState_(obj, which, newState)` — parameterized state machine

The previous Phase 1027 single-pane version (~100 lines) was replaced with a parameterized version (~115 lines) that:

1. Validates `which` ∈ `{'events','live'}` (new error ID `FastSenseCompanion:invalidLogWhich`) and `newState` ∈ `{'Inline','Detached','Hidden'}`.
2. Dispatches to per-pane locals: `pane`, `dd`, `panel`, `detName` (the property *name* string for dynamic access), `figTitle`, `figSize`.
3. Idempotency check: derives current state from `pane.IsAttached` + `obj.(detName)` validity (NOT from the dropdown — same Phase 1027 lesson as commit `3e6c155` because dropdowns are pre-set during bootstrap).
4. Branches: `'Inline'` → tear down det fig if any (clear `CloseRequestFcn` first), `pane.detach()` if attached, `pane.attach(panel, Theme_)`. `'Detached'` → `pane.detach()`, build new `uifigure` with em-dash title via `sprintf(..., char(8212))` and per-pane size, set `CloseRequestFcn = @(~,~) obj.setLogState_(which, 'Inline')` (closes over `which`), `pane.attach(newFig, Theme_)`. `'Hidden'` → tear down det fig if any, `pane.detach()`.
5. Syncs `dd.Value = newState`, calls `obj.rebalanceLogStrip_()`.
6. Wraps the whole branch in try/catch + `uialert` so a state-transition failure never crashes the companion.

### `rebalanceLogStrip_` — single source of truth

New private helper (~30 lines) that:

- Computes `evtInline = EventsLogPane_.IsAttached && (hDetachedEventsFig_ empty/invalid)`. Same for `liveInline`.
- Sets inner `hLogStripGrid_.RowHeight`:
  - both inline: `{180, '1x'}`
  - events only: `{'1x', 0}`
  - live only:   `{0, '1x'}`
  - neither:     `{0, 0}`
- Sets outer `hLayout_.RowHeight{3}` to `OriginalLogRowHeight_` if at least one inline, else `0`.

This is the single function that mutates either row-height; `setLogState_` no longer touches them directly.
